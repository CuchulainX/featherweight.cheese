To print a lifesize SVG, use Firefox on Linux to produce a PDF, then
use Adobe Reader to send to the printer. For Firefox, Print > Print to
File > Page Setup > Paper size: Tabloid > Options > Uncheck "Ignore
Scaling and Shrink To Fit Page Width" > set everything to --blank-- in
Header and Footer. For Adobe Reader, set the paper size to Tabloid,
maximize the print resolution, and turn off any options for rescaling.

To print the source code as a pamphlet, use enscript. 
enscript --indent=10 --fancy-header --highlight=python \
  --media=tabloid --portrait --nup=2 --color=1 --nup-xpad=72 \
  --nup-ypad=72 --style=emacs --underlay="Cheese" *.py -p foo.ps
